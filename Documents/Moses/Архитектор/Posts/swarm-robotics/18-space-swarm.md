# Глава 18. Космический рой (формации спутников и межпланетные системы)

## Зачем эта глава

Космический рой — самый масштабный и самый «далёкий» от земной инженерии случай. Здесь рой оперирует в вакууме, при экстремальных температурах, с задержками связи в секунды-минуты, без возможности физического обслуживания. Каждый аппарат стоит миллионы, запуск — десятки миллионов, потеря — необратима. И при этом именно в космосе рой даёт максимальный выигрыш: один тяжёлый спутник не масштабируется, а тысяча мелких — да.

Эта глава — про распределённые космические системы, формации, группировки и межпланетные рои.

## 1. Особенности космической среды

### 1.1 Физика

| Параметр | Значение | Следствие |
|---|---|---|
| Вакуум | 10⁻⁹ Па (LEO) — 10⁻¹⁵ Па (межзвёздное) | Нет конвективного охлаждения, спекание материалов |
| Температура (LEO) | −150…+150 °C | Перепады при входе/выходе из тени |
| Температура (глубокий космос) | −270 °C | Криогенные системы |
| Микрогравитация | 10⁻⁶ g (LEO) — 0 g | Нет «верха» и «низа», поведение отличается |
| Радиация | Высокая (Van Allen belts) | Электроника требует защиты |
| Микрометеориты | 1–10 м/с в LEO, 70 км/с гиперболические | Угроза целостности |
| Атмосферное торможение (LEO) | Заметно на 200–500 км | Спутник тормозит, нужен поддерживающий манёвр |
| Световое давление | 9 мкН/м² | Заметная сила на большие площади |
| Альбедо Земли | 30–40% солнечной энергии | Освещение с разных сторон |
| Задержка связи (LEO) | 5–20 мс | Почти real-time |
| Задержка связи (GEO) | 240 мс × 2 = 0,5 с | Real-time диалог ещё возможен |
| Задержка связи (Луна) | 1,3 с × 2 = 2,6 с | Real-time диалог невозможен |
| Задержка связи (Марс) | 4–24 мин × 2 = 8–48 мин | Только store-and-forward |
| Задержка связи (Юпитер) | 33–53 мин × 2 = 1–2 ч | Абсолютно асинхронно |
| Задержка связи (межзвёздная) | годы | Только полная автономия |

### 1.2 Следствия для дизайна

- **Связь** — радио или оптика, но **с задержкой**, растущей с расстоянием.
- **Сервис невозможен** — отказавший спутник не починить (кроме особых случаев).
- **Коррекция орбиты** — ограничена запасом топлива или электрической тягой.
- **Энергия** — солнечные панели (внутри орбиты Марса), радиоизотопные (дальше) или ядерный реактор.
- **Теплоотвод** — только излучением (радиаторы).
- **Стоимость** — на порядки выше подводных систем.

## 2. Орбиты и группировки

### 2.1 Типы орбит

| Орбита | Высота | Период | Назначение |
|---|---|---|---|
| LEO (Low Earth Orbit) | 200–2000 км | 90–120 мин | Дистанционное зондирование, IoT, МКС, Starlink |
| MEO (Medium Earth Orbit) | 2000–35 786 км | 2–24 ч | GPS, ГЛОНАСС, Galileo |
| GEO (Geostationary) | 35 786 км | 24 ч | Связь, метео |
| HEO (Highly Elliptical) | переменная | переменная | Связь в высоких широтах (Тундра, Молния) |
| SSO (Sun-Synchronous) | 600–800 км, полярная | специальная | Разведка, метео |
| Cislunar | вокруг точки L1/L2 система Земля-Луна | — | Космические телескопы |
| Lunar orbit | 100–500 км над Луной | — | Лунные миссии |
| Martian orbit | 200–2000 км над Марсом | — | Mars Reconnaissance |
| Heliocentric | разные | — | Межпланетные миссии |

### 2.2 Кеплеровы элементы орбиты

Шесть параметров, описывающих орбиту:
- **a** — большая полуось (размер).
- **e** — эксцентриситет (форма).
- **i** — наклонение (угол к экватору).
- **Ω** — долгота восходящего узла.
- **ω** — аргумент перицентра.
- **ν** — истинная аномалия (положение на орбите).

Для роя важно, чтобы все спутники имели **согласованные** элементы, иначе они быстро «разбегутся».

### 2.3 Относительные орбиты (formation flying)

Спутники в рое летают не по независимым орбитам, а с **малым отклонением** от опорной. Используются относительные орбитальные элементы (ROE — Relative Orbital Elements) для описания взаимного положения.

Типичные формации:
- **Кольцо (ring)** — все спутники на одной орбите, разнесены по периоду.
- **Train (string-of-pearls)** — на одной орбите, разнесены по аномалии.
- **Cartwheel** — круговая формация в плоскости, перпендикулярной движению.
- **Projected circular** — круг, проецируемый на Землю.
- **Pendulum** — два спутника качаются относительно друг друга (типа GRACE).
- **3D formation** — произвольная пространственная структура.

## 3. Архитектуры космического роя

### 3.1 Walker constellation (Δ-constellation)

Классическая архитектура, описанная Джоном Уокером (1970):
- **T** спутников на **P** плоскостях, по **S** спутников в каждой.
- Все плоскости с одинаковым наклонением **i**.
- Плоскости равномерно распределены по долготе восходящего узла.
- Спутники в соседних плоскостях сдвинуты по фазе (phase difference **F**).

Обозначается **T/P/F** (например, 24/3/1 — 24 спутника в 3 плоскостях по 8, сдвиг 1).

**Примеры:**
- **GPS**: 24/6/1 (исходный), сейчас 31+ спутник.
- **Galileo**: 24/3/1.
- **Iridium NEXT**: 66/6/11 (66 спутников в 6 плоскостях по 11, сдвиг 11).
- **Starlink Gen1**: 4408 в нескольких оболочках (Shell 1: 72 плоскости, по 22).

### 3.2 Streets-of-coverage

Спутники распределены по «улицам» — группам соседних орбитальных плоскостей. Улица обеспечивает непрерывное покрытие заданной полосы широт.

### 3.3 Sparse и dense formations

- **Sparse** — спутники на расстоянии км-десятков км (типа Sentinel-1A/B).
- **Dense** — спутники на расстоянии метров (технологические демонстраторы).
- **Tight** — десятки-сотни метров (космические интерферометры).

### 3.4 Federated satellite systems

Спутники **от разных владельцев**, объединённые в общую систему через стандарты. Пример: метеорологические спутники разных стран обмениваются данными.

### 3.5 Fractionated spacecraft

Несколько **отдельных** аппаратов, совместно выполняющих функцию одного. Преимущества:
- отказ одного — не катастрофа,
- легче выводить на орбиту,
- проще обновлять (заменить один, не трогая другие).

**DARPA System F6** (2013) — программа по фракционированным спутникам, завершена без развёртывания, но идеи живут.

## 4. Связь в космическом рое

### 4.1 Межспутниковая связь (ISL — Inter-Satellite Links)

Оптическая или радио-связь между спутниками без участия наземной станции.

- **Оптическая ISL** — лазер, до 10 Гбит/с, узкая диаграмма, требует точного наведения.
- **Радио ISL** — до 100 Мбит/с (S, Ka, V диапазоны), проще в наведении.

**Starlink** — каждый спутник имеет 4 лазерных ISL (с 2022+), что позволяет передавать данные между спутниками по цепочке без наземных станций.

### 4.2 Связь с Землёй

- **TT&C (Telemetry, Tracking, Command)** — узкий канал, обычно S- или X-диапазон, для управления.
- **Downlink** — основной канал данных (X, Ka, оптический).
- **Uplink** — для команд (редко используется в рое с автономностью).
- **GNSS** — спутники используют GPS/ГЛОНАСС/Galileo для орбитальных измерений.

### 4.3 DTN — Delay/Disruption Tolerant Networking

Протокол для связи с большими задержками и перерывами. Использует **store-and-forward**: пакеты хранятся на узлах до тех пор, пока связь не появится.

**Bundle Protocol (BP)** — основной протокол DTN, стандартизирован IETF (RFC 9171).

Применяется в:
- связи с марсоходами,
- межпланетном интернете,
- связи с подводными лодками,
- космическом рое с большими расстояниями.

### 4.4 P2P в космосе

Каждый спутник может быть и клиентом, и маршрутизатором. Это **mesh-сеть в космосе**:
- пакеты идут кратчайшим путём через роутинг в рое,
- каждый спутник хранит таблицы маршрутизации,
- топология меняется (спутники движутся) — нужна адаптивная маршрутизация.

## 5. Определение относительного положения (GNSS-relative)

### 5.1 GPS как основной инструмент

Спутники в LEO/GEO используют GNSS-приёмники для **абсолютной** навигации (точность 1–10 м). Для **относительной** навигации (взаимное положение) нужна точность сантиметры-миллиметры.

### 5.2 Differential GNSS (DGNSS)

Использует **дифференциальные поправки** от наземной станции с известной позицией. Точность: 10 см — 1 м.

### 5.3 Real-Time Kinematic (RTK)

Несущая фаза измеряется, точность 1–10 см в real-time. Требует плотной сети базовых станций.

### 5.4 Precise Point Positioning (PPP)

Использует точные эфемериды и поправки часов спутников GNSS. Точность 5–20 см без базовой станции.

### 5.5 Радиоинтерферометрия

- **Same-beam interferometry** — два спутника в одной диаграмме наземной станции измеряют разность фаз. Точность миллиметры.
- **VLBI** (Very Long Baseline Interferometry) — разность фаз между двумя наземными станциями. Используется для геодезии, не для роёв напрямую.

### 5.6 Star trackers

Оптические датчики по звёздам. Точность 1–10 угл.сек. Используются для **взаимной** ориентации спутников.

## 6. Алгоритмы управления формацией

### 6.1 Управление с обратной связью (feedback control)

Классическая теория управления, адаптированная к космосу. Цикл: «измерить позицию → сравнить с желаемой → выдать команду двигателям».

Алгоритмы:
- **PID** — простейший, работает в стабильной формации.
- **LQR (Linear Quadratic Regulator)** — оптимальный для линейных систем.
- **H-infinity** — робастный, учитывает неопределённости.
- **MPC (Model Predictive Control)** — предсказывает траекторию, оптимизирует на горизонте.

### 6.2 Согласованное управление (coordinated control)

Несколько спутников согласовывают свои действия через:
- **Centralized MPC** — один «лидер» решает за всех, остальные исполняют.
- **Distributed MPC** — каждый решает за себя, согласовывая через обмен информацией.
- **Consensus-based** — спутники сходятся к общему решению итеративно (BFT / консенсус).

### 6.3 Управление с учётом возмущений

Возмущения в космосе:
- **Атмосферное сопротивление** (LEO) — требует регулярной коррекции.
- **Солнечное давление** — особенно важно для больших сплеш-зон.
- **Гравитационные аномалии** (J2, J3 и т.д.) — отличие Земли от идеальной сферы.
- **Лунно-солнечные приливы** — небольшие, но накапливаются.
- **Magnetic drag** — взаимодействие с магнитным полем Земли.

Возмущения нужно **измерять** (акселерометры, GPS) и **компенсировать** (двигателями).

### 6.4 Fuel-optimal maneuvers

Задача: перевести рой из одной формации в другую с минимальным расходом топлива. Это:
- задача выпуклой оптимизации (для линеаризованной динамики),
- задача нелинейной оптимизации (для точной модели),
- перебор эвристик (для больших размерностей).

### 6.5 Collision avoidance

- **Conjunction assessment** — оценка сближений с другими объектами.
- **Maneuver planning** — расчёт манёвра уклонения.
- **Cooperative avoidance** — два роя согласовывают, кто уклоняется.

### 6.6 Orbit determination (OD)

Точное знание орбиты критично. Используются:
- **Фильтр Калмана** (extended, unscented) — стандарт.
- **Batch least squares** — точный, но не real-time.
- **Стохастические модели возмущений** — для предсказания.

## 7. Реальные системы и группировки

### 7.1 Связь

#### Starlink (SpaceX)
- **4000+** спутников в LEO (на 2025), планируется 30 000+.
- **Несколько оболочек** (shell): 540–570 км, разные наклонения.
- **Лазерные ISL** с 2022 — данные передаются между спутниками.
- **Цель** — глобальный интернет с задержкой 20–50 мс.

#### OneWeb
- **648** спутников в LEO (1200 км).
- **Полярная** группировка.
- **Только Earth-pointed** — без ISL пока.

#### Kuiper (Amazon)
- **3236** спутников в LEO.
- **Запуск** начат в 2024, развёртывание продолжается.

#### Guowang (Китай)
- **~13 000** спутников в LEO (заявлено).
- **Государственная** программа, военное и гражданское использование.

#### Qianfan / Spacesail (Китай)
- **~14 000** спутников в LEO (заявлено).
- Альтернатива Starlink для Китая.

#### Глобальные навигационные системы
- **GPS (США)** — 31 спутник в MEO.
- **ГЛОНАСС (Россия)** — 24 спутника в MEO.
- **Galileo (ЕС)** — 24+ спутника в MEO.
- **BeiDou (Китай)** — 35+ спутников в MEO/GEO/IGSO.
- **IRNSS/NavIC (Индия)** — 7 спутников в GEO/IGSO.
- **QZSS (Япония)** — 4+ спутника в HEO/GEO.

### 7.2 Дистанционное зондирование

#### Copernicus (ESA)
- **Sentinel-1A/1B/1C** — радарный интерферометр (C-band).

#### Landsat (NASA/USGS)

#### Maxar, BlackSky, Satellogic, ICEYE, Capella
- Десятки компаний с коммерческими группировками.

#### NASA Earth Observing System
- Традиционные крупные спутники (Terra, Aqua, Aura, Suomi NPP).
- Сейчас переход к **коммерческим** данным.

### 7.3 Научные миссии

#### GRACE / GRACE-FO (NASA/DLR)
- **Два спутника** на одной орбите, расстояние ~220 км.
- Измеряют **вариации гравитационного поля** Земли.
- Применения: подземные воды, таяние ледников, тектоника.

#### TanDEM-X (DLR)
- **Два радарных спутника** (TerraSAR-X + TanDEM-X) в плотной формации.
- Построили **глобальную ЦМР** с разрешением 12 м.

#### Swarm (ESA)
- **Три спутника** для изучения **магнитного поля** Земли.
- Летят в разных плоскостях.

#### Proba-3 (ESA, запуск 2025)
- **Два аппарата**, летящие в формации с точностью **1 мм** на расстоянии 150 м.
- Создают **искусственное солнечное затмение** для наблюдения короны.

### 7.4 Военные и разведывательные

#### NRO (США)
- Спутники-шпионы детально не публикуются.
- **NROL**, **KH-11**, **Lacrosse/Onyx** — известные программы.

#### SBIRS / OPIR (США)
- **Геостационарные** и **высокоэллиптические** спутники раннего предупреждения о ракетном нападении.
- Сенсоры **инфракрасные** — обнаружение горячих факелов ракет.

#### Russian military
- **«Космос»** серия — десятки спутников.
- **«Тундра»** — система раннего предупреждения нового поколения.

#### Yaogan (Китай)
- **30+** спутников радиолокационной и оптической разведки.

### 7.5 Межпланетные

#### Mars Cube One (MarCO) (NASA, 2018)
- **Два кубсата 6U** — первые кубсаты в межпланетном пространстве.
- Лете вместе с InSight, передавали данные с InSight при посадке.

#### Lunar Pathfinder (ESA, 2025+)
- **Лунный** коммуникационный ретранслятор.

#### Lunar Communications Relay and Navigation Systems (LCRNS)
- **NASA** — группировка для обслуживания лунных миссий.

#### Marconi / Cooperative Autonomous Distributed Explorative (CADE)
- Концепт **роя межпланетных кубсатов** для разведки астероидов.

## 8. Технологические вызовы

### 8.1 Стоимость запуска

Исторически запуск стоил $10 000–20 000 за кг (Shuttle, Ariane 5). Сейчас — $1000–3000/кг (Falcon 9), прогноз на 2030 — $100–500/кг (Starship).

Снижение стоимости → больше спутников → рост в формациях.

### 8.2 Миниатюризация

- **Cubesat** — 1U = 10×10×10 см, 1–1,3 кг.
- **6U, 12U, 16U** — большие кубсаты.
- **Smallsat** — от 10 до 500 кг.
- **Traditional** — от 500 кг.

Рой из 1000 кубсатов массой 1 кг = 1 тонна. Это **одна** миссия.

### 8.3 Стандартизация

- **CubeSat Standard** (Cal Poly) — стандарт на размер и интерфейсы.
- **SpaceWire, SpaceFibre** — сетевые протоколы для космоса.
- **CCSDS** (Consultative Committee for Space Data Systems) — стандарты NASA/ESA.
- **Open Mission Stack** — открытый стек (NASA).

### 8.4 Радиационная защита

- **Rad-hard компоненты** — дорого, длинный цикл разработки.
- **COTS (Commercial Off-The-Shelf)** — дешевле, но требует экранирования и redundancy.
- **TID (Total Ionizing Dose)** — доза, после которой компонент отказывает.
- **SEE (Single Event Effects)** — одиночные сбои (bit flip, latch-up).

### 8.5 Терморегулирование

- **MLI (Multi-Layer Insulation)** — основная изоляция.
- **Heaters** — для поддержания температуры в тени.
- **Radiators** — для сброса тепла.
- **Heat pipes** — для распределения тепла.

### 8.6 Двигатели

#### Химические
- **Моногидразин** (N₂H₄) — стандарт, токсичен.
- **AF-M315E** — «зеленое» топливо, нетоксично.
- **Биперекись водорода (H₂O₂)** — нетоксично.
- **Твёрдотопливные** — для больших манёвров, не дросселируются.

#### Электрические
- **Ионные (Xenon)** — малый удельный импульс, долгий разгон.
- **Hall-effect** — то же, проще конструктивно.
- **Электромагнитные (PPT, VAT)** — твёрдое рабочее тело.
- **VASIMR** (плазменный) — высокий импульс, экспериментальный.

#### Без рабочего тела
- **Solar sail** — световое давление.
- **Gravity assist** — бесплатно, но требует точной траектории.
- **Tether** — электродинамический.

## 9. Спутники-инспекторы и обслуживание

### 9.1 Mission Extension Vehicle (MEV) (Northrop Grumman)

- **MEV-1** (2020) — состыковался с Intelsat-901 на GEO и продлил его жизнь на 5 лет.
- **MEV-2** (2021) — аналогично с Intelsat-10-02.
- **MRV** — будущие миссии по дозаправке и ремонту.

### 9.2 Restore-L (NASA, отменён)

Планировался для захвата и обслуживания спутника **Landsat 7**.

### 9.3 OSAM-1, OSAM-2 (NASA)

Программы по **орбитальному обслуживанию**: заправка, перестановка компонентов, сборка в космосе.

### 9.4 ClearSpace (ESA)

**ClearSpace-1** (запуск 2026) — первый европейский миссия по **удалению космического мусора**. Захватит фрагмент Vespa (верхняя ступень Vega) и сведёт его с орбиты.

### 9.5 Astroscale (Япония)

- **ELSA-d** (2021) — демонстратор захвата и увода.
- **ELSA-M** (2025+) — коммерческая услуга по удалению мусора.
- **ADRAS-J** (2024) — сближение с фрагментом H-IIA.

### 9.6 RemoveDEBRIS (Surrey Space Centre)

Демонстратор с **сетью** и **гарпуном** для захвата мусора. Успешно провёл испытания (2018–2019).

## 10. Удаление космического мусора (ADR — Active Debris Removal)

### 10.1 Проблема

На орбите **> 36 000** объектов > 10 см (отслеживаемых) и **> 1 000 000** объектов 1–10 см (не отслеживаются). Скорость столкновения — 7–15 км/с (кинетическая энергия как у пули, но в 10–20 раз больше). Синдром Кесслера — каскад столкновений, делающий орбиту непригодной.

### 10.2 Стратегии

- **Захват и сведение** — робот захватывает, уводит с орбиты, сгорает в атмосфере.
- **Лазерное удаление** — наземный или орбитальный лазер сбивает фрагмент с орбиты.
- **Буксир** — сцепка с фрагментом, спуск в плотные слои.
- **Сеть/гарпун** — пассивный захват.
- **Уничтожение** — фрагмент дробится на более мелкие, которые сгорят быстрее.
- **Deorbit sail** — парус, увеличивающий сопротивление и ускоряющий сход с орбиты (только для активных аппаратов).

### 10.3 Юридические проблемы

- **Кому принадлежит мусор?** Запустившая страна несёт ответственность, но не обязана убирать.
- **Кто платит?** Затраты на ADR — миллиарды. Кто оплачивает?
- **Страхование** — мусор может повредить коммерческие спутники. Вопросы суброгации.

## 11. Космические интерферометры

### 11.1 Принцип

Несколько **отдельных** телескопов в формации, **интерферирующих** как один виртуальный телескоп. Эквивалентное фокусное расстояние = расстояние между аппаратами.

### 11.2 Реальные и планируемые

- **LISA** (ESA, запуск 2035) — **три** аппарата в вершинах равностороннего треугольника со стороной 2,5 млн км. Регистрация **гравитационных волн**.
- **DARPA TPF-I** (Terrestrial Planet Finder — Interferometer) — концепт поиска экзопланет, не реализован.
- **Hypertelescope** (Labeyrie) — распределённый оптический телескоп с тысячами малых апертур.
- **Cosmic Vision** (ESA) — различные концепты.

### 11.3 Вызовы

- **Точность позиционирования** — пикометры (10⁻¹² м) на миллионах км.
- **Лазерная метрология** — измерение расстояний между аппаратами.
- **Drag-free** — каждый аппарат защищён от внешних сил.
- **Синхронизация** — часы на аппаратах с точностью 10⁻¹⁸.

## 12. Лунные и окололунные системы

### 12.1 Lunar Communications Relay (LCRNS)

NASA планирует **группировку спутников** на лунной орбите для:
- связи с экипажами Artemis,
- навигации (Lunar GNSS),
- поисково-спасательных операций.

### 12.2 ESA Moonlight

Проект **создания группировки** для связи и навигации на Луне. Партнёры: ESA, NASA, JAXA, Surrey Satellites.

### 12.3 Lunar Gateway

Окололунная **космическая станция** (NASA, ESA, JAXA, CSA). Не рой, но компонент.

### 12.4 Лунный интернет

Концепт **mesh-сети** на лунной поверхности — ретрансляторы, обеспечивающие связь везде, включая **кратеры** и **полюса** (где наземные ретрансляторы не помогут).

## 13. Межпланетные рои

### 13.1 CubeSats в межпланетном пространстве

- **MarCO** (NASA, 2018) — первые межпланетные кубсаты. Успех.
- **Lunar Flashlight** (NASA) — поиск водяного льда (не достиг орбиты, но концепт верен).
- **CAPSTONE** (NASA, 2022) — кубсат 12U на лунной орбите, тест NRHO.
- **Hera Milani** (ESA, 2025+) — кубсат 6U, полетит к астероиду Didymos после DART.
- **LICIACube** (ASI, 2022) — кубсат для наблюдения за DART-ударом.

### 13.2 Distributed spacecraft для межпланетных миссий

- **ST-5** (NASA, 2006) — три микроспутника для измерения магнитного поля Земли.
- **MMS** (NASA, 2015) — **четыре** аппарата в тетраэдре для изучения **магнитосферы**.
- **Cluster** (ESA, 2000) — **четыре** аппарата в тетраэдре для изучения **магнитосферы**.
- **THEMIS** (NASA, 2007) — **пять** аппаратов в разных орбитах для изучения **суббурь**.

### 13.3 Asteroid mining (будущее)

Рой аппаратов для разведки и добычи на астероидах. Преимущества:
- один аппарат не справляется с разведкой,
- рой распределяет риск,
- модульный подход.

Экономическая целесообразность под вопросом, но технологически возможно.

### 13.4 Зонды фон Неймана (см. главу 14)

Самовоспроизводящийся рой — самый эффективный способ колонизировать галактику. Не реализован и не близок к реализации.

### 13.5 Breakthrough Starshot (2016+)

Проект **лазерного разгона** микрозондов (чип на парусе) к **Альфа Центавра** на 20% скорости света. Длительность перелёта — 20–30 лет.

- Наноспутник массой **1 грамм** с лазерным парусом.
- Лазерная батарея на Земле разгоняет парус.
- Связь с Землёй — лазерная, задержка 4+ года.

Пока на стадии концепта, но с серьёзным финансированием ($100M+ от Юрия Мильнера и др.).

## 14. Этические и правовые вопросы

### 14.1 Договор о космосе (1967)

- **Outer Space Treaty** — основа международного космического права.
- Космос — **достояние человечества**, не подлежит национальному присвоению.
- Запрет оружия массового уничтожения в космосе.
- Государства несут ответственность за национальные космические объекты.

### 14.2 Проблемы

- **Kessler syndrome** — каскад столкновений, угрожающий всем.
- **Starlink и астрономия** — тысячи спутников мешают наземным наблюдениям.
- **Light pollution** — спутники ярче звёзд для астрономии.
- **Radio pollution** — передатчики мешают радиоастрономии.
- **Space debris** — см. §10.
- **Orbital slots** — GEO переполнен, нужно международное регулирование.
- **Spectrum allocation** — кто и как использует частоты.
- **Военное использование** — рои в космосе как оружие.

### 14.3 Mega-constellations и регулирование

- **FCC** (США) — лицензирование Starlink, Kuiper, OneWeb.
- **ITU** — распределение частот.
- **UN COPUOS** — Комитет ООН по космосу.
- **IADC** — Межагентский координационный комитет по мусору.
- **LTS Guidelines** — руководство ООН по долгосрочной устойчивости.

### 14.4 Права на орбиты

GEO — уникальный ресурс. Слоты распределяются **первым пришёл — первым обслужен**. Без международного регулирования — конфликты.

### 14.5 Контакт с внеземным разумом

Если рой зондов встретит инопланетный рой — что делать? Это уже обсуждается в литературе (Search for Extraterrestrial Intelligence — SETI). METI (Messaging) — попытки передать сигнал в космос. Активно критикуется за возможные последствия.

## 15. Кейсы и сценарии

### 15.1 Сценарий 1: Глобальный интернет через мегагруппировку

- **Задача:** обеспечить интернет в любой точке Земли.
- **Архитектура:** тысячи спутников в LEO, лазерные ISL, связь с абонентскими терминалами.
- **Примеры:** Starlink, Kuiper, OneWeb.
- **Риски:** световое загрязнение, столкновения, конкуренция за орбиты и частоты.
- **Будущее:** интеграция с terrestrial 5G/6G, IoT, прямое подключение абонентских устройств.

### 15.2 Сценарий 2: Мониторинг климата

- **Задача:** непрерывный глобальный мониторинг температуры, CO₂, ледников, океана.
- **Архитектура:** рой спутников с разными сенсорами (радар, лидар, спектрометр).
- **Примеры:** Sentinel, Sentinel-6, GRACE-FO, ICESat-2.
- **Будущее:** десятки малых спутников вместо одной большой миссии.

### 15.3 Сценарий 3: Лунная база и Artemis

- **Задача:** обеспечить связь и навигацию экипажей Artemis на Луне.
- **Архитектура:** группировка на лунной орбите + ретрансляторы на поверхности.
- **Примеры:** LCRNS, Moonlight, Lunar Pathfinder.
- **Будущее:** 10+ лунных спутников, обслуживающих постоянную базу.

### 15.4 Сценарий 4: Марс и его спутники

- **Задача:** исследование Марса, связь с поверхностью.
- **Архитектура:** орбитальные аппараты + ретрансляторы.
- **Примеры:** Mars Reconnaissance Orbiter, ExoMars TGO, Mars Express, Tianwen-1.
- **Будущее:** группировка для **3D-картографии** Марса, поиск воды, подготовка к пилотируемой миссии.

### 15.5 Сценарий 5: Спасение от астероидов

- **Задача:** обнаружить и отклонить опасный астероид.
- **Архитектура:** рой спутников для раннего обнаружения + рой кинетических ударников.
- **Примеры:** DART (NASA, 2022) — успешный удар по Dimorphos. Hera (ESA) — изучение последствий.
- **Будущее:** готовая группировка кинетических перехватчиков на «дежурной» орбите.

## 18. Итог

Космический рой — это **финальный рубеж** для роевых технологий. Если подводный рой — это «тёмная лошадка», то космический — это **венец**, сочетающий:
- минимальный размер агента (кубсат 1U),
- максимальное расстояние (межпланетное и межзвёздное),
- экстремальные условия (радиация, вакуум, температура),
- уникальные применения (наблюдение Земли, связь, наука, оборона, межпланетные миссии).

Самые горячие направления сегодня:
- **mega constellations** для глобального интернета,
- **ADR** для спасения орбиты от мусора,
- **formation flying** для космических интерферометров,
- **interplanetary CubeSats** для разведки Солнечной системы,
- **orbital servicing** для продления жизни спутников.

Через 30–50 лет, возможно, у нас будет **рой из тысяч кубсатов**, обслуживающих **лунную базу**, **марсианскую колонию**, и работающих в **межпланетной mesh-сети**. Это не фантастика — это **проектируемая инженерная реальность**.
